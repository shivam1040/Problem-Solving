1. This application is made using Spring Boot. Import in IDE(preferably STS or Eclipse) and run the application, and ensure the availability of port 8080 to use the endpoints.

2. API for single word response is, localhost:8080/api/word1/word2

3. API for multi word response is, localhost:8080/api1/word1/word2

4. Data caching and optimisation is done while application is initiallized.
